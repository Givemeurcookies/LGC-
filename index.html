What is LGC+? LGC is a simple file sharer, created for The Gathering 2013. The files are hosted on a file server with 6TB of file space.


Encrypted transferring, we're running the server on an encrypted SSL protocol, meaning people won't be able to observe your file transfers.


Chat, we decided to add a chat, because why not? You can easily use the client to contact anyone else using LGC+, get suggestions, talk dirty, whatever you like.


Stream, yes, you can stream any file you like, every file is accessible using a browser or whatever you'd like on port 80. Meaning you can use for example VLC's network streaming to stream videos or even music.


Cross platform, we both got OSX and Windows, sadly, we don't have support for Linux generally.
